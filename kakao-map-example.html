카카오맵 API 설정 가이드

  1. 카카오디벨로퍼스에서 앱 생성
    developers.kakao.com에서 앱 생성 완료 ✓
  2. JavaScript 키 발급
    앱 설정 > 플랫폼 > Web 플랫폼 등록 > JavaScript 키 확인
    사이트 도메인에 http://localhost 또는 실제 도메인 등록
  3. 카카오맵 사용 설정
    앱 관리 페이지 > 카카오맵 > 사용 설정 > 상태: ON
  4. 아래 코드의 YOUR_JAVASCRIPT_KEY를 실제 키로 변경
⚠️ 주의:
• 반드시 JavaScript 키를 사용하세요 (REST API 키 아님)
• 프로덕션 환경에서는 API 키를 노출하지 않도록 주의하세요
